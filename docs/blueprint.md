# **App Name**: Gym Diary

## Core Features:

- Workout Input Form: Simple form for daily workout data entry.
- Workout Table: Display workout data in a clear, sortable table.
- Monthly Summary Calculation: Calculate monthly workout summary and totals for key metrics like total weight lifted, total time spent running.
- Responsive Layout: Display workout data, including text and calculated summaries, optimized for both desktop and mobile devices.
- Workout Data Extraction: Use natural language processing (NLP) to extract structured workout data (exercise name, weight, reps, sets, running time, max heart rate) from the user's free-text input using an LLM tool.
- Local Data Storage: Save workout data locally using browser local storage.

## Style Guidelines:

- Primary color: A calming blue (#64B5F6) to evoke a sense of stability and focus during workouts.
- Background color: A light, desaturated blue (#E3F2FD) to create a clean and airy feel.
- Accent color: An energizing orange (#FFB74D) to highlight key elements and calls to action.
- Body text font: 'PT Sans' for a clear and readable presentation of workout data.
- Headline font: 'Space Grotesk' for a modern and slightly technical feel.
- Use clear, minimalist icons to represent different exercises and workout metrics.
- Prioritize a clean, tabular layout for easy data entry and review, optimized for mobile responsiveness.
